## DeepSeek++ 0.7.1

0.7.1 是工具调用和上下文注入稳定性版本，重点让长回复、可下载产物、记忆和 Skill 在同一条任务链里更稳地协作。

### 主要变化

- **工具执行更稳定**：长回复中的工具调用可以更早、更稳地被识别和执行，减少工具块残留在正文、重复触发或等待完整回复结束才处理的情况。
- **可下载产物更顺畅**：单文件和项目包产物生成过程更适合流式执行，长内容生成时页面更容易保持可交互。
- **记忆与 Skill 上下文更可靠**：记忆、Skill 和系统提示词在多步续跑和历史整理中更准确地进入任务上下文，减少上下文丢失或重复注入。
- **回复正文更干净**：工具调用内容与普通 Markdown 输出分离更清楚，用户看到的是更干净的回复正文和工具执行结果。
- **回归覆盖继续补强**：新增和补强工具解析、流式文本、历史清理、页面执行和产品表面的自动化覆盖，降低长任务回归风险。
- **发布质量继续收口**：构建工具依赖归回开发依赖，生产依赖审计恢复为 0 漏洞；发布资产校验继续覆盖 Chrome、Edge、Firefox 和源码包。
