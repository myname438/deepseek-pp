## DeepSeek++ 0.4.2

0.4.2 是发布准备和隐私展示增强版本，重点补齐 Chrome Web Store 提交材料，并避免内部提示词与工具格式提醒在 DeepSeek 页面或历史记录中暴露。

### 主要变化

- 新增 Chrome Web Store 上架材料，包括商店文案、隐私政策、提交流程、权限说明、审核说明和必需截图资产。
- 新增 Chrome Web Store 上传 workflow 和本地提交脚本，支持 dry-run、仅上传和上传后提交审核三种路径。
- 页面和历史记录会清理内部提示词标记与工具格式提醒，只保留用户可见提示和可读工具执行结果。
- 流式回复解析更严格区分真实回复文本、思考内容和非回复事件，减少内部上下文误参与工具调用解析的情况。
- 将 Chrome Web Store 凭据环境文件加入忽略列表，避免本地提交密钥被误提交。
- README 新增 0.4.2 变更回顾，并继续保留 0.4.1 / 0.4.0 / 0.3.0 / 0.2.0 的折叠历史。
