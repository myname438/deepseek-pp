## DeepSeek++ 0.3.0

0.3.0 将 DeepSeek++ 从 Chrome 单目标扩展推进到 Chrome / Edge / Firefox 多浏览器发布，同时补齐深浅色体验、版本一致性和多浏览器 release 资产。

### 主要变化

- 新增 Chrome、Edge、Firefox MV3 构建与打包脚本：`build:chrome`、`build:edge`、`build:firefox`、`build:all`、`zip:chrome`、`zip:edge`、`zip:firefox`、`zip:all`。
- WXT manifest 改为按目标浏览器生成：Chrome/Edge 保留 `sidePanel`，Firefox 生成兼容侧栏入口和稳定 Gecko ID。
- Release workflow 改为执行 `zip:all` 并上传所有浏览器 zip，不再只发布单个 Chrome 包。
- 侧边栏顶部导航重做为稳定 tab 组件，改善窄侧栏下的图标、文字和当前页状态。
- 新增 DeepSeek 页面主题检测和同步，侧边栏、Skill 弹窗、工具执行卡片、工具结果区块和自定义背景遮罩都适配明暗主题。
- 设置页、记忆页、MCP 页的颜色和状态样式改为主题 token，减少浅色硬编码。
- 顶部版本徽标、设置页底部版本和 MCP clientInfo 改为从扩展 manifest 读取版本，修复仍显示 0.1.0 的问题。
- MCP Native Messaging、安装说明和验证文档改为浏览器中立表述，补充 Chrome / Edge / Firefox 的加载入口。
- 归档 MCP rollout 文档，并新增 Edge/Firefox 支持归档，保留验证记录和手动测试线索。

### 验证

- `npm run compile`
- `npm run build:all`
- `npm run zip:all`
- `npm run smoke:mcp`
- `npm run verify:mcp:mock`
