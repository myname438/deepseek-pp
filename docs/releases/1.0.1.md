## DeepSeek++ 1.0.1

1.0.1 是 1.0.0 之后的体验和可靠性补丁，重点让会话速度、项目会话和工具执行边界更清楚、更可控。

### 主要变化

- **用量与速度可见性**：设置页新增用量统计入口，可查看 token、请求、耗时和速度等本地统计，帮助用户理解当前会话和工具链路的执行节奏。
- **工具执行边界更清楚**：MCP 工具执行和多模态能力检查更严格，减少未启用、未配置或不适合当前环境的工具被误触发。
- **项目会话体验更稳**：项目侧栏里的会话链接和主题样式继续打磨，深色/浅色主题下的列表、链接和状态更清楚。
- **商店素材补齐**：补充 Chrome Web Store 使用的图标和宣传图，让商店页面能更准确展示当前产品界面。
- **回归覆盖继续补强**：新增 token 速度、用量统计、MCP 执行策略、多模态策略、侧边栏导航和项目侧栏整理测试。
